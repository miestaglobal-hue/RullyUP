# 🎾 RallyUp Bot

Telegram-бот для поиска партнёров по теннису и паделу.

## Структура файлов

```
rallyup_bot/
├── bot.py              # Точка входа
├── config.py           # Токен, константы
├── database.py         # SQLite — все запросы
├── keyboards.py        # Клавиатуры
├── requirements.txt
└── handlers/
    ├── __init__.py
    ├── menu.py         # /start, профиль, скрыть
    ├── register.py     # Регистрация (FSM)
    ├── swipe.py        # Свайп карточек + инвайт
    └── invite.py       # Принять / отклонить инвайт
```

## Запуск за 5 шагов

### 1. Получи токен
Напиши @BotFather → /newbot → получи токен вида `7123456789:AAF...`

### 2. Установи зависимости
```bash
cd rallyup_bot
pip install -r requirements.txt
```

### 3. Задай токен
**Вариант A — переменная окружения (рекомендую):**
```bash
export BOT_TOKEN="твой_токен_здесь"
```

**Вариант B — прямо в config.py:**
```python
BOT_TOKEN = "7123456789:AAF..."
```

### 4. Запусти
```bash
python bot.py
```

### 5. Напиши боту /start

---

## Деплой на Railway (бесплатно)

1. Зарегайся на [railway.app](https://railway.app)
2. New Project → Deploy from GitHub repo
3. В Variables добавь: `BOT_TOKEN = твой_токен`
4. Add → Python → Deploy

Бот будет работать 24/7 бесплатно.

## Флоу пользователя

```
/start
  └─► Регистрация (имя → пол → спорт → уровень → город → время → фото)
        └─► Главное меню
              ├─► 🎾 Искать партнёра
              │     └─► Выбор: кого ищешь? (парней / девушек / всех)
              │           └─► Свайп карточек
              │                 ├─► ❌ Пропустить → следующая карточка
              │                 └─► ✅ Позвать → инвайт кандидату
              │                           └─► Кандидат: принять / отклонить
              │                                 └─► Матч → контакты открываются
              ├─► 👤 Мой профиль
              ├─► ✏️ Изменить профиль
              └─► 🔕 Скрыть меня
```
